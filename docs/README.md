# Documentation Index

`docs/FOUNDERS-BLUEPRINT.md` is the canonical product specification.

All other markdown files in this directory are subordinate references:

- `ARCHITECTURE.md`
- `DECISIONS.md`
- `IMPLEMENTATION-BLUEPRINT.md`
- `IMPLEMENTATION-CHECKLIST.md`
- `ROADMAP.md`

Rules for the documentation set:

- The blueprint defines product authority.
- Derived docs may explain, sequence, or record decisions.
- Derived docs must not override the blueprint.
- New strategy docs should not be added unless explicitly requested.
